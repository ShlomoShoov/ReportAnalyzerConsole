# Report Analyzer Console App

This app Analyze a reports fils and create an orgnize report about the file data.

### ***How to run This APP***
In The project folder

`dotnet run`

### ***optinal flags***

***add all flags after --***

***`-f fileName`*** Change the source file

***`-o fileName`*** Define an output file

***`--debug`*** Cancel the debug messages (they wont display in the terminal)
